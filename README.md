Number Guessing Game 🎯
A simple interactive number guessing game built using HTML, CSS, and JavaScript.
The game generates a random number between 1 and 100, and the player tries to guess it.
🚀 Features
Random number generation between 1 and 100
User input field to enter guesses
Button to submit guesses
Feedback after every guess:
Too High
Too Low
Correct Guess
Displays the total number of attempts
Option to restart the game
🛠️ Technologies Used
HTML
CSS
JavaScript
